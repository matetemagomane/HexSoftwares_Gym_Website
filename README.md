HexSoftwares_Gym_Website
Project Overview
This project is part of my Web Development Internship at Hex Softwares. The objective was to create a modern, single-page gym website that showcases facilities, services, and membership plans in a scrollable format.

Features

Single-Page Navigation: Smooth scrolling between different sections of the site.

Hero Section: High-impact landing area to grab user attention.


Facilities & Services: Organized grids highlighting what the gym offers.


Membership Plans: Clear pricing tiers for potential members.


Responsive Design: Built to work across different screen sizes using modern CSS.

Technologies Used

HTML5: For the structure of the web pages.


CSS3: For styling, layout (Flexbox/Grid), and animations.
